# Raw Datasets

This directory contains original, unmodified datasets used for evaluating
physical reasoning in large language models.

## Included datasets
1. **PIQA** (Physical Interaction QA)
   A well known dataset for physical commonsense reasoning. it tests everyday physical intuition and good baseline for intuitive physics tasks.
2. **CLEVRER**
   Causal reasoning, object interaction and counterfactuals physical reasoning
3. **PHYRE**
   Physical interaction and outcome prediction
